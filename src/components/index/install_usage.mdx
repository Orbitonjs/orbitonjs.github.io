### Installation

Using a CDN

```html
<script
  crossorigin
  src="https://unpkg.com/orbiton@latest/umd/orbiton.production.js"
></script>
```

Using Npm or yarn

```bash
# yarn
yarn add orbiton

# npm
npm install orbiton
```

</Card>

### Usage

Try it on [Codesandbox]()

```js
import Orbiton from "orbiton";

const Button = (
  <button
    onClick={() => {
      alert("Clicked");
    }}
  >
    Click me
  </button>
);

Orbiton.append(Button, document.getElementById("root"));
```
